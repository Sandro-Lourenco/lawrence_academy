# Create Feature

Use o workflow `../../workflows/new-feature.md`. Produza primeiro um contrato de feature com problema, atores, regras, permissões, escopo, fora de escopo, métricas, riscos e aceite. Implemente em fatias verticais pequenas; mantenha rastreabilidade requisito -> código -> teste -> documentação. Não avance sobre decisão de produto ambígua nem ação de alto risco sem aprovação.
